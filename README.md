# Bomberman HTML5

A simple Bomberman clone made with HTML5, JavaScript, and CSS. Playable in your browser.

## How to Play
- Use the arrow keys to move your character.
- Press the spacebar to place a bomb.
- Destroy all enemies to win.
- Avoid enemy contact and bomb explosions!

## Files
- index.html: Main HTML file
- main.js: Game logic
- style.css: Game styles

## To Run
Just open index.html in your browser.
